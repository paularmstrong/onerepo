## Writing plugins

Plugins are objects comprised of three possible hooks.

| Key                          | Type                                | Description                                                                                                                                      |
| ---------------------------- | ----------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| [`startup`](/api/#startup)   | `(argv) => Promise<void> \| <void>` | Run a function _before_ any and all commands, after argument parsing.                                                                            |
| [`shutdown`](/api/#shutdown) | `(argv) => Promise<void> \| <void>` | Run a function _after_ any and all commands, after argument parsing.                                                                             |
| [`yargs`](/api/#yargs)       | `(yargs, visitor) => Yargs`         | A function that is given the CLI's [Yargs instance](https://yargs.js.org/docs/#api-reference). Useful for adding custom commands and middleware. |
